# Psi-Sensor
PSI sensor is a hardware/software device that has several different sensors that provide data that may be used to identify possible paranormal activity. It is intended to be used in Home Assistant as an ESPHome device and its sensors can be used to influence the personality of an AI assistant capable of using sensors to affect it.   [Barabashka](https://github.com/KeithSBB/barabashka) is another poject that integrates Grok AI with Home assistant and in its configuration you can select sensors (ex: all the PSI sensor measurements) to modify Grok's behavior.
